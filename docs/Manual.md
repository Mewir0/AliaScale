###  ルールベース編集の独自変数

ルールベース編集では名前生成に以下の変数を使用できる。

| 変数 | 意味 |
| --- | --- |
| `$p` | prefix。末尾の空白を含む |
| `$m` | 発音 |
| `$pm` | prefix + 発音 |
| `$s` | suffix |
| `$a` | 旧エイリアス |
| `$f` | 旧wav名の拡張子を取ったもの |
| `$n` | 推定または丸め後の音階名 |
| `$l` | Preview上の行番号 |
| `$mr` | Preview上の同一発音内番号 |
| `$pmr` | Preview上の同一prefix+発音内番号 |
| `$d` | 重複名に付ける番号 |
